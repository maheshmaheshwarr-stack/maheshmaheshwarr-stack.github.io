# Mahesh Ramanathan - Personal Website

A clean, modern personal website showcasing projects, about section, and contact information.

## Features

- **Responsive Design** - Works beautifully on all devices
- **Modern Aesthetic** - Clean, professional design with smooth animations
- **Fast & Lightweight** - Pure HTML/CSS/JS, no frameworks
- **SEO Optimized** - Proper meta tags and semantic HTML
- **Smooth Scrolling** - Elegant navigation between sections

## Sections

1. **Hero** - Introduction and call-to-action
2. **Projects** - Showcase of your work (Mahesh +ve Pill featured)
3. **About** - Your story and background
4. **Contact** - Multiple ways to connect

## Customization

### Update Your Information

1. **Contact Details** (lines 520-545):
   - Replace email: `your.email@example.com`
   - Update LinkedIn URL
   - Update Twitter handle

2. **About Section** (lines 480-500):
   - Customize your bio
   - Add your photo or keep the emoji

3. **Projects** (lines 430-475):
   - Add more project cards
   - Update descriptions and links

### Color Scheme

The site uses CSS variables for easy theming (lines 18-24):
```css
--primary: #6366f1;        /* Main brand color */
--primary-dark: #4f46e5;   /* Darker shade */
--text: #0f172a;           /* Main text */
--text-muted: #64748b;     /* Secondary text */
```

## Deployment

### GitHub Pages
1. Create a new repository (e.g., `maheshramanathan.github.io`)
2. Push this file as `index.html`
3. Enable GitHub Pages in repository settings
4. Your site will be live at `https://maheshramanathan.github.io`

### Custom Domain
1. Add a `CNAME` file with your domain
2. Configure DNS settings with your domain provider
3. Enable HTTPS in GitHub Pages settings
